## Project Description
This project is developed as part of the Advanced Programming Laboratory course.  
It is a simple space shooter game built using the **C programming language** in **Visual Studio**.  
The player controls a spaceship,avoids enemies and shoots obstacles to gain score.


## Features
- Player-controlled spaceship
- Enemy generation
- Shooting mechanism
- Score tracking
- Level up
